### Arrays 
Arrays is a collection of similar items in a contiguous memory location.
#### Array operations 
- Delete : removes element from the array 
- Add : insert a new element into the array
- sort : Arranges the array in a soecified order
- Traverse : moving through the elements of the array 

The disadvantages to arrays is their inefficiencies at larger operations 

### Linked List
A linked list is a collection of arrays in a non contiguous memory location. it has a node which consist of data and a pointer to the next node. There are two special node in a linked list -- the HEAD node and the TAIL node. the last node of a linked list points to NULL.

#### Types of linked list
- singly linked list: a pointer that points to the next node only 
- Double linked list: a pointer points to the next node and another points to the previous node
- Circular linked list: the HEAD node is linked to the TAIL node

#### Advantages (double )
- Bidirectional Taversal
- Efficient Deletion
- Insertion at both ends 
- Easy to implement Deque

#### Disadvantages 
- Extra memory per node
- More Complex Implementation
- not cache friendly
